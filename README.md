# kneg
kneg is a simple todo list manager for the command line.  

```
$ kneg -a "My first task"
$ kneg
    #1 | My first task
$ kneg -a "My second task"
$ kneg
    #1 | My first task
    #2 | My second task
$ kneg -c 1
$ kneg
    #2 | My second task
```
